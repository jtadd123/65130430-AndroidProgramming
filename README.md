## Hướng dẫn sử dụng
1. Mở **Android Studio**.
2. Chọn **Open** và dẫn đến thư mục của từng bài tập cụ thể.
3. Chờ Gradle đồng bộ (Sync) hoàn tất.
4. Nhấn nút **Run** (phím tắt Shift + F10) để chạy trên thiết bị giả lập hoặc thật.
