## Tong quan du an

Repository gom nhieu mini app Android, moi thu muc la mot bai thuc hanh rieng biet.

- Tang dan do kho tu giao dien co ban den xu ly su kien va danh sach du lieu.
- Tap trung vao nen tang can ban: `Activity`, `Intent`, `ListView`, `OnClick`, `Toast`.
- Phu hop de on tap, review code nhanh va so sanh cach trien khai qua tung bai.

## Huong dan chay

### 1) Clone repository

```bash
git clone https://github.com/jtadd123/65130430-AndroidProgramming.git
```

### 2) Mo bang Android Studio

- Chon **Open**
- Tro den thu muc repository vua clone
- Doi Android Studio sync Gradle xong

### 3) Chay tung mini app

- Vao thu muc app mong muon (vi du: `AppBMI` hoac `TemperatureConverterApp`)
- Chon emulator hoac ket noi thiet bi that
- Bam **Run**
